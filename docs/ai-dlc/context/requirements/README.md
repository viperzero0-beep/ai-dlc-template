# 要件コンテキスト（Requirements）

Inception フェーズの成果物（要件・ユーザーストーリー・受入基準・Unit of Work）を保存します。

- 命名: `docs/ai-dlc/context/requirements/<unit-of-work>.md`（kebab-case）
- 生成: `/inception` プロンプトを使用
- これらは Construction フェーズの**入力**になります

## ファイルに含める内容

- ビジネス意図 / 解決する課題
- ユーザーストーリー（`〜として、〜したい、なぜなら〜`）
- 受入基準（Given / When / Then）
- Unit of Work の Bolt への分解
- リスク・前提・スコープ外
- Construction への引き継ぎ事項 / 未解決の意思決定
