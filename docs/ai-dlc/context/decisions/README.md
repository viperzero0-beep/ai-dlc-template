# 意思決定記録（ADR: Architecture Decision Records）

重要な技術的意思決定を**連番**で記録します。背景・決定・理由・トレードオフ・却下案を残し、
「なぜそう決めたか」を将来の自分・チーム・AI が辿れるようにします。

- 命名: `NNNN-<kebab-title>.md`（`NNNN` は 4 桁ゼロ埋めの連番）
- 生成: `/adr` プロンプトを使用
- ステータス: `提案中` → `承認済み`（人間の承認後）→ 必要なら `置き換え済み(→ NNNN)`
